# FacebookUnfriend
Unfriend inactive users


## Install
      $ pkg install php
      $ git clone https://github.com/fajarprasetiya/fb-unfriend
      $ cd fb-unfriend

## Run
      $ php unfriend.php

## Updating
      $ git pull --force

## Explanation :
- Input :
    - Facebook token `your facebook token (xxxxx)`
    - Year `Ex : 2018, unfriend users where doesn't update status before 2018`
